# File Explorer Written in C

This File Explorer was made because frankly, searching files in any file explorer just sucks.
The windows one takes forever and ones which are fast dont even explore the folder much.

This File explorer will cache directory structure into a cache file and then it will search that file 
for text matches to then reconstruct the path.

The previous one, written in C++ worked, except the fact that its completely unusable,
so I'm rewriting it in C, with ui inspired from other file explorers
